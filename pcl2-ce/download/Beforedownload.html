下载前准备工作 ​

下载加速器

提示：

请确保您的设备符合下面的要求 ​

  • 设备操作系统为： Windows。
  • Windows 版本为：Windows 10 20H2 （Build 19042） 或更高版本。
  • .NET 版本要求：.NET Framework 4.8.1 或以上。

如果您可以直接打开 这个网站 请直接跳转到下一章 手动下载。 ​

1.下载加速器 ​

1.1打开 Watt Toolkit 官网,点击这里，steam++ 官网会自动识别设备是 ARM64 或 AMD64 架构。
1.2点击下载 Windows 版。

1.3 点击 接受并下载。

1.4 蓝奏云网盘。
1.5 输入密码 1234。
1.6 下载软件。
请选择 Steam++_v x.x.x-rc.15_win_xxx.exe 下载。

2.安装加速器+使用 ​

2.1 双击运行 Steam++_v x.x.x-rc.15_win_xxx.exe

2.2 点击立即安装
2.3 安装完成后，点击立即体验 2.4 打开后点击 网络加速 ， 勾线github。
2.5 点击一键加速
2.6 测试是否能访问这个网站，如果不行请尝试使用 一键下载
